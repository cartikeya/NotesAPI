# Notes API

A simple REST API for managing notes using:

- Node.js
- Express.js
- MongoDB

## Setup

1. Clone the repository

```bash 
git clone (https://github.com/cartikeya/NotesAPI)
cd NotesAPI
```
2. Install dependencies
```bash
npm install
```
3. Make sure MongoDB is running locally
```bash
mongodb://localhost:27017
```
4. Start the server
```bash
node app.js
```
